# Sys_Snapshot_AI

**An intelligent AI-powered PC & Laptop Specification Extractor**

Turn your DxDiag report into a **beautiful, interactive HTML system profile** with one click.

![Version](https://img.shields.io/badge/version-6.0-blue)
![Python](https://img.shields.io/badge/Python-3.8%2B-brightgreen)
![Platform](https://img.shields.io/badge/Platform-Windows-blue)

---

### ✨ Features

- **Smart GPU Detection** — Automatically identifies Dedicated (NVIDIA/AMD) vs Integrated (Intel) GPUs
- **5 Beautiful Tabs**:
  - Overview
  - GPU & Display
  - Ports & I/O (with model-specific port layouts)
  - Performance (Gaming FPS estimates + benchmark bars)
  - System Details + Full Raw Data
- Supports **.txt** and **.docx** DxDiag files
- Generates **professional HTML reports** with modern cyber-tech design
- Local web interface with drag & drop
- Auto-saves reports in `~/DxDiag_Outputs/`
